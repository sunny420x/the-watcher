# What is the watcher ?
The watcher is web-based network scanner tool helping discovering services such as web server, ip camera, ftp and ssh in globle IPv4 range.

# Discovering building in dark forest is kinda.. fun!
In the watcher, you can scan services from all over the world ! it's like walking through the jungle and discovering abandoned building but in computer network !
